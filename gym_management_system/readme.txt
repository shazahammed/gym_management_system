# Gym-Management-System
DBMS mini project on Gym Management

A simple dynamic website for notes aggregation by students
Made by using - PHP, MySQL, CSS, Javascript

Use Xampp to host this project

Go to XAMPPs MySql server and create a database named 'project_db'

Import the sql file from datadase folder

enter url 'localhost/gym_management_system/index.html'
